# Design System Documentation

## Theme Overview

This document outlines the core visual characteristics of our design system.

## Color Palette

Our color palette is designed to be vibrant and distinct.

-   **Primary Color:** `#050dff` - This is our brand's most distinctive color, used for key interactive elements and calls to action.
-   **Secondary Color:** `#656dd3` - A supporting color for less prominent UI elements and secondary actions.
-   **Tertiary Color:** `#9f0002` - An additional accent color for highlights, badges, or decorative elements.
-   **Neutral Color:** `#767684` - A foundational neutral for backgrounds, surfaces, and non-chromatic elements.
-   **Color Mode:** `light` - The system operates in a light mode theme.

## Typography

We use a consistent font family across our interface for clarity and brand recognition.

-   **Headline Font:** `inter` - Used for all prominent titles and headings.
-   **Body Font:** `inter` - Applied to all main text content.
-   **Label Font:** `inter` - Used for form labels, small annotations, and interactive element labels.

## Shape and Spacing

Our design maintains a balanced approach to shape and spatial relationships.

-   **Roundedness:** `2` (Moderate) - UI elements feature a moderate level of rounded corners, providing a friendly yet professional aesthetic.
-   **Spacing:** `2` (Normal) - A normal level of whitespace is maintained, offering a comfortable balance between information density and readability.
